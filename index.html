CodeIgniter-Subqueries by NTICompass

An active record subquery library for CodeIgniter.

Adds subquery capabilities to CodeIgniter's Active Record class.

Dependencies

PHP5 CodeIgniter 1.7.x or 2.x

Install

Put Subquery.php into application/libraries, then load it in your code.
You can add subquery to $autoload['libraries'] in application/config/autoload.php, or load it by calling $this->load->library('Subquery').
